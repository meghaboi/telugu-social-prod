# Phase 1 Launch Gates

## Product Gates

- Onboarding completion rate and funnel tracked.
- Pulse shows exactly 15 cards with 5/5/5 source grouping.
- Registered event pinned card expires 48h after event end.
- Spaces/forums usable with nested replies and moderation actions.
- Free event registration successful end-to-end.

## Policy and Safety Gates

- Report/block/mute flows fully functional.
- Staff can triage moderation queue.
- Audit logging enabled for staff-sensitive operations.

## Engineering Gates

- Zero open Sev-1/Sev-2 defects.
- Crash-free sessions and API error budgets within target.
- P95 latency within agreed threshold for key endpoints.

## Release Gates

- iOS build distributed via TestFlight internal testing.
- Android build distributed via Play internal testing.
- Web user app and staff dashboard deployed on production infra.
- Rollback and incident response playbook validated.
